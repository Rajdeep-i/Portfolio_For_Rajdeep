INFINITY SHIFT

About This Game

I make this game For a Game Jam. Infinity Shift is a endless game theme base game. In this game jam they gave us theme, theme is endless/Infinity. I develop this game in 3 day as a solo developer. In this game a space ship navigate throughout the space. avoid / destroy all the obstacles using missiles and movement. When player doing is thing he/she gat a Survival points. In this game achievement is player how long survive. and when player collide with obstacles game is over.

My role: I doing Total work, like I doing Game Design, and Game Programming etc

Tools/Technologies used: In this game Tools uses are unity as a game engine, Visual Studio as a programming editor. Platform is Windows.
